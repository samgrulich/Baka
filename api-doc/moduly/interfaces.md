# Modul interfaces
Vrací podporované verze API/rozhraní

## XML
### Příklad
```xml
<?xml version="1.0" encoding="utf-8"?>
<interfaces>
  <interface>
    <version>1</version>
    <deprecated>false</deprecated>
    <experimental>false</experimental>
  </interface>
  <result>01</result>
</interfaces>
```
